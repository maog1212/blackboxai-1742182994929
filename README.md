# 左右互搏术练习程序

基于金庸武侠小说中"左右互搏术"概念的互动练习游戏

Built by https://www.blackbox.ai

## 📦 项目内容

本项目包含三个版本的左右互搏术练习程序：

### 🖥️ Python桌面版
- **文件**: `zuoyou_hubo.py`
- **说明**: `ZUOYOU_HUBO_README.md`
- **适用**: Windows/Mac/Linux桌面环境
- **运行**: `python3 zuoyou_hubo.py`

### 📱 移动Web版（基础版）
- **文件**: `zuoyou_hubo_mobile.html`
- **适用**: 📱 iPhone/iPad (Safari/Chrome) | Android
- **使用**: 粘贴网址即可，无需安装
- **特点**: 轻量级，快速加载，完美支持iOS

### 🚀 移动Web版（增强版）⭐ 推荐iPhone用户
- **文件**: `zuoyou_hubo_enhanced.html`
- **适用**: 📱 iPhone (Safari ✅ / Chrome ✅) | Android Chrome
- **使用**: 粘贴网址即玩，可选开启手势识别
- **特点**:
  - ✅ iPhone Safari/Chrome 完美兼容
  - 🎥 可选手势识别功能
  - 📷 所有功能已内置，无需安装任何东西
  - 🤖 一键开启AI交互

## 🎮 游戏特色

- 🎭 4个经典武侠角色（郭靖、小龙女、杨过、周伯通）
- 🥋 14种经典武功招式（降龙十八掌、六脉神剑等）
- ⚔️ 完整的招式相克系统
- 🎯 4个难度等级
- 📊 详细的数据统计
- 📈 熟练度评级系统
