# 左右互搏术练习程序

基于金庸武侠小说中"左右互搏术"概念的互动练习游戏

Built by https://www.blackbox.ai

## 📦 项目内容

本项目包含两个版本的左右互搏术练习程序：

### 🖥️ Python桌面版
- **文件**: `zuoyou_hubo.py`
- **说明**: `ZUOYOU_HUBO_README.md`
- **适用**: Windows/Mac/Linux桌面环境
- **运行**: `python3 zuoyou_hubo.py`

### 📱 移动Web版
- **文件**: `zuoyou_hubo_mobile.html`
- **说明**: `iOS_使用说明.md`
- **适用**: iPhone/iPad/Android手机
- **运行**: 在Safari或Chrome浏览器中打开

## 🎮 游戏特色

- 🎭 4个经典武侠角色（郭靖、小龙女、杨过、周伯通）
- 🥋 14种经典武功招式（降龙十八掌、六脉神剑等）
- ⚔️ 完整的招式相克系统
- 🎯 4个难度等级
- 📊 详细的数据统计
- 📈 熟练度评级系统

## 🚀 快速开始

### 桌面版
```bash
python3 zuoyou_hubo.py
```

### 移动版
1. 在iPhone上用Safari打开 `zuoyou_hubo_mobile.html`
2. 或访问GitHub Pages链接（如已部署）
3. 添加到主屏幕以获得最佳体验
