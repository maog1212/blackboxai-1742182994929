# 左右互搏术练习程序

基于金庸武侠小说中"左右互搏术"概念的互动练习游戏

Built by https://www.blackbox.ai

## 📦 项目内容

本项目包含三个版本的左右互搏术练习程序：

### 🖥️ Python桌面版
- **文件**: `zuoyou_hubo.py`
- **说明**: `ZUOYOU_HUBO_README.md`
- **适用**: Windows/Mac/Linux桌面环境
- **运行**: `python3 zuoyou_hubo.py`

### 📱 移动Web版（基础版）
- **文件**: `zuoyou_hubo_mobile.html`
- **说明**: `iOS_使用说明.md`
- **适用**: iPhone/iPad/Android手机
- **运行**: 在Safari或Chrome浏览器中打开
- **特点**: 轻量级，快速加载

### 🚀 移动Web版（增强版）⭐ 推荐
- **文件**: `zuoyou_hubo_enhanced.html`
- **说明**: `OCR_手势识别说明.md`
- **适用**: Chrome/Safari浏览器，支持摄像头的设备
- **运行**: 在浏览器中打开，授予摄像头权限
- **特点**:
  - ✅ Chrome完美优化
  - 🎥 实验性手势识别
  - 📷 OCR集成准备
  - 🤖 支持MediaPipe/TensorFlow.js

## 🎮 游戏特色

- 🎭 4个经典武侠角色（郭靖、小龙女、杨过、周伯通）
- 🥋 14种经典武功招式（降龙十八掌、六脉神剑等）
- ⚔️ 完整的招式相克系统
- 🎯 4个难度等级
- 📊 详细的数据统计
- 📈 熟练度评级系统

## 🚀 快速开始

### 桌面版
```bash
python3 zuoyou_hubo.py
```

### 移动版（基础版）
1. 在iPhone/Android上打开 `zuoyou_hubo_mobile.html`
2. 或访问GitHub Pages链接（如已部署）
3. 添加到主屏幕以获得最佳体验

### 移动版（增强版）⭐
1. 在Chrome或Safari中打开 `zuoyou_hubo_enhanced.html`
2. 点击"手势识别模式"开关启用摄像头
3. 授予摄像头权限
4. 用手势控制招式选择
