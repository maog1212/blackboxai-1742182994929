# 左右互搏术练习程序

## 📖 项目介绍

这是一个基于金庸武侠小说中"左右互搏术"概念的互动练习程序。在程序中，你可以扮演不同的武侠角色，体验左手和右手同时运行不同武功招式的奇妙感觉！

## ✨ 功能特色

### 🎭 角色系统
- **郭靖（北侠）**：基础实力90，擅长降龙十八掌
- **小龙女（古墓仙子）**：基础实力88，擅长玉女心经
- **杨过（神雕侠）**：基础实力92，擅长黯然销魂掌
- **周伯通（老顽童）**：基础实力95，擅长左右互搏

| 招式名称 | 类型 | 威力 | 描述 |

### ⚔️ 招式相克系统

```
拳 → 克制 → 掌、棍
掌 → 克制 → 指、剑
指 → 克制 → 拳、内功
剑 → 克制 → 拳、棍
内功 → 克制 → 掌、剑
棍 → 克制 → 指、内功
```

### 🎮 难度等级
- **初学（Easy）**：10回合练习
- **熟练（Normal）**：20回合练习
- **精通（Hard）**：30回合练习
- **宗师（Master）**：50回合练习

### 📊 进度系统
- **左右手协调度**：随练习提升
- **熟练度评级**：从初学者到宗师
- **战斗统计**：左手获胜、右手获胜、势均力敌
- **进步曲线**：可视化显示练习成果

## 🚀 使用方法

### 运行程序

```bash
python3 zuoyou_hubo.py
```

### 操作流程

1. **选择角色**：输入1-4选择你想扮演的角色
2. **选择难度**：输入1-4选择练习难度
3. **开始练习**：按回车键开始每一回合
4. **查看结果**：每回合显示左右手对战结果和协调度提升
5. **完成练习**：查看详细统计数据和熟练度评级

## 🎯 游戏目标

通过不断练习，提升角色的左右手协调度和熟练度，最终达到"宗师级"境界！

## 💡 设计亮点

1. **真实的武侠体验**：14种经典武功招式，还原金庸武侠世界
2. **策略性相克系统**：不同招式类型之间存在相克关系
3. **成长系统**：协调度和熟练度会随练习不断提升
4. **随机性与平衡**：每回合随机选择招式，威力有随机波动
5. **视觉反馈**：ASCII艺术让文字界面更加生动有趣
6. **完善的统计**：详细的数据统计帮助了解练习成果

## 🔧 技术实现

- **面向对象设计**：Move、Character、Game类清晰分离
- **类型注解**：使用Python类型提示增强代码可读性
- **模块化代码**：功能模块清晰，易于扩展
- **用户体验**：流畅的交互流程和即时反馈

## 📈 熟练度等级

- 🌱 **0-30%**：初学者 - 继续努力练习
