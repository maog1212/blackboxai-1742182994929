# 📱 iPhone快速入门指南

## ⚡ 3分钟上手

### 🎯 最简单的方法（推荐）

#### 步骤1：获取文件
从以下任一方式获取 `zuoyou_hubo_mobile.html` 文件：

```
方式A：AirDrop
1. 在Mac/PC上找到文件
2. 右键 → 共享 → AirDrop
3. 选择你的iPhone
4. 在iPhone上接收

方式B：邮件
1. 将文件作为附件发送到你的邮箱
2. 在iPhone上打开邮件
3. 点击附件下载

方式C：云盘
1. 上传到iCloud Drive/百度网盘
2. 在iPhone上下载
```

#### 步骤2：用Safari打开
```
1. 在iPhone上找到下载的文件
2. 长按文件
3. 选择"分享" 📤
4. 选择"在Safari中打开"
```

#### 步骤3：添加到主屏幕（可选但推荐）
```
1. 在Safari中打开文件后
2. 点击底部分享按钮 📤
3. 向下滚动
4. 点击"添加到主屏幕"
5. 点击"添加"
6. 现在主屏幕上会出现游戏图标！
```

完成！现在你可以像打开App一样玩游戏了！🎉

## 🌐 使用GitHub Pages（更推荐）

### 一次部署，永久使用

如果你有GitHub账号，这是最佳方式：

#### 在电脑上操作：

```bash
# 1. 确保文件已经推送到GitHub
git push origin main

# 2. 在GitHub仓库页面：
#    - 点击 Settings（设置）
#    - 左侧找到 Pages
#    - Source选择 main分支
#    - 点击 Save
#    - 等待几分钟

# 3. 你会获得一个URL，类似：
# https://你的用户名.github.io/仓库名/zuoyou_hubo_mobile.html
```

#### 在iPhone上操作：

```
1. 打开Safari浏览器
2. 输入你的GitHub Pages URL
3. 添加书签或添加到主屏幕
4. 随时随地都能玩！
```

**优势：**
- ✅ 不需要下载文件
- ✅ 自动更新到最新版本
- ✅ 可以分享给朋友
- ✅ 任何设备都能访问

## 💡 游戏技巧

### 理解相克系统
```
拳 克 掌、棍
掌 克 指、剑
指 克 拳、内功
剑 克 拳、棍
内功 克 掌、剑
棍 克 指、内功
```

### 提升熟练度
- **左右平衡最重要**：势均力敌时协调度+5（最多）
- **单方胜利**：协调度+3
- **多玩几轮**：熟练度=协调度÷10
- **目标90%**：达到宗师级！

### 选择角色
- **郭靖（90）**：平衡型，适合新手
- **小龙女（88）**：柔中带刚
- **杨过（92）**：攻击力强
- **周伯通（95）**：最强实力，挑战高难度

## ❓ 常见问题

## 📞 需要帮助？

如果遇到问题：

1. **重新加载页面**
   - 在Safari中向下拉刷新

2. **清除缓存**
   - 设置 → Safari → 清除历史记录与网站数据

3. **重启Safari**
   - 双击Home键（或上滑）关闭Safari
   - 重新打开

4. **重启设备**
   - 最后的解决方案
