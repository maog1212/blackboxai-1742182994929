## 快速开始

### 步骤 1：获取 API Key

1. 访问 [DeepSeek 平台](https://platform.deepseek.com/)
2. 注册或登录您的账号
3. 进入 API Keys 页面
4. 创建新的 API Key 并复制

### 步骤 2：配置应用

1. 打开应用后，点击右下角的 **⚙️ 设置按钮**
2. 在弹出的设置面板中粘贴您的 API Key
3. 点击 **保存设置**

### 步骤 3：开始对话

现在您可以开始使用了！试试这些话题：

#### 🎯 目标设定
```
我想在2025年实现以下目标：
1. 学习一门新技能
2. 提升职业能力
3. 改善健康状况
请帮我制定一个详细的计划
```

#### 💼 职业规划
```
我目前是一名初级程序员，
想在3年内成长为高级工程师，
请给我一些建议和学习路线
```

#### 📚 学习计划
```
我想学习人工智能相关知识，
但不知道从哪里开始，
请帮我制定一个6个月的学习计划
```

#### ⚖️ 生活平衡
```
最近工作很忙，感觉生活失衡，
如何更好地平衡工作和个人生活？
```

## 功能详解

### 1. 智能对话

助手会根据您的问题提供个性化建议：

- **理解上下文**：记住之前的对话内容
- **结构化回复**：使用清晰的格式组织建议
- **可操作建议**：提供具体的执行步骤

### 2. 快捷按钮

界面下方提供了常用话题的快捷按钮：

| 按钮 | 功能 |
|------|------|
| 📚 学习计划 | 快速开始制定学习计划 |
| 💼 职业规划 | 获取职业发展建议 |
| ⚖️ 生活平衡 | 讨论工作生活平衡 |
| 🎯 目标设定 | 设定和规划人生目标 |

### 3. 对话历史

- **自动保存**：对话会自动保存在浏览器中
- **持续对话**：刷新页面后可以继续之前的对话
- **清空记录**：点击 🗑️ 按钮清空所有对话

### 4. 设置管理

- **API Key 管理**：随时更新您的 API Key
- **本地存储**：所有设置仅保存在您的设备上
- **隐私保护**：不会上传到任何服务器

## 常见问题

### Q1: API Key 是否安全？

**A:** 是的。您的 API Key 仅保存在浏览器的 LocalStorage 中，不会发送到除 DeepSeek 官方服务器外的任何地方。

### Q2: 对话记录保存在哪里？

**A:** 对话记录保存在浏览器的 LocalStorage 中。清除浏览器数据会删除这些记录。

### Q3: 如何清空对话历史？

**A:** 点击右下角的 🗑️ 清空对话按钮，确认后即可清空所有对话记录。

### Q4: 是否需要付费？

**A:** 应用本身免费，但需要 DeepSeek API 的使用费用。请查看 DeepSeek 官网的定价信息。

### Q5: 支持哪些浏览器？

**A:** 支持所有现代浏览器：Chrome、Firefox、Safari、Edge 等。

### Q6: 可以在手机上使用吗？

**A:** 可以！应用采用响应式设计，完美支持移动设备。

### Q7: 为什么 API 调用失败？

可能的原因：
- API Key 不正确
- API 额度用完
- 网络连接问题
- DeepSeek 服务暂时不可用

解决方法：
1. 检查 API Key 是否正确
2. 查看 DeepSeek 控制台的使用情况
3. 检查网络连接
4. 稍后重试

### Q8: 如何获得更好的回复？

提供更多上下文信息：
```
✅ 好的提问：
"我是一名5年经验的前端工程师，
想转型做全栈开发，
目前对后端技术了解不多，
请帮我制定一个学习计划"

❌ 不够清晰的提问：
"我想学后端"
```

## 最佳实践

### 1. 提供详细背景

告诉助手：
- 您的当前状况
- 您的目标
- 您的限制条件（时间、资源等）
- 您已经尝试过的方法

### 2. 分步骤规划

对于复杂目标，可以分阶段讨论：

```
第一步：总体规划
"我想在5年内实现财务自由，
请帮我分析可行性和大致方向"

第二步：细化计划
"基于刚才的建议，
请帮我制定第一年的详细计划"

第三步：执行跟踪
"我执行计划已经1个月了，
遇到了一些问题，想调整一下..."
```

### 3. 定期回顾

建议每月或每季度与助手回顾进展：

```
"上个月我们制定了学习计划，
现在我已经完成了前两个阶段，
但在第三阶段遇到了困难，
请帮我调整计划"
```

### 4. 利用快捷功能

- 使用快捷按钮快速开始话题
- 善用 Enter 键快速发送
- Shift+Enter 进行换行

### 5. 保持对话连贯

由于助手会记住对话历史，您可以：
- 继续之前的话题
- 引用之前的建议
- 要求深入某个方面

## 小贴士

1. **定期使用**：建议每周至少与助手对话一次，保持规划的连续性
2. **记录进展**：可以截图保存重要建议，配合笔记工具使用
3. **诚实沟通**：遇到困难或偏离计划时，及时与助手讨论调整
4. **设定提醒**：为重要的行动计划设置日历提醒
5. **分享反馈**：执行建议后，告诉助手效果如何，以便获得更好的后续建议
