## ⚡ 超快部署（GitHub Pages）

### 方法一：自动部署（推荐）

**只需1分钟，获得永久网址！**

#### 步骤1：启用GitHub Pages

1. 打开你的GitHub仓库页面
2. 点击 **Settings**（设置）
3. 在左侧菜单找到 **Pages**
4. 在 **Source** 下选择：
   - Branch: `main` (或当前分支)
   - Folder: `/ (root)`
5. 点击 **Save**

#### 步骤2：等待部署（约30秒）

GitHub会自动部署，完成后显示：
```
✅ Your site is live at https://你的用户名.github.io/仓库名/
```

#### 步骤3：获取游戏网址

你的游戏网址是：

**基础版：**
```
https://你的用户名.github.io/仓库名/zuoyou_hubo_mobile.html
```

**增强版（推荐）：**
```
https://你的用户名.github.io/仓库名/zuoyou_hubo_enhanced.html
```

#### 步骤4：分享给用户

**在iPhone上（Safari或Chrome）：**
1. 粘贴网址
2. 开始游戏
3. 添加到主屏幕（可选）

✅ **完成！** 用户无需任何操作，直接玩！

## 🎯 iPhone用户使用指南

### 1️⃣ 收到网址后

**在Safari中：**
```
1. 长按网址 → 拷贝
2. 打开Safari
```

```
1. 复制网址
2. 打开Chrome
3. 粘贴到地址栏
```

### 2️⃣ 添加到主屏幕（推荐）

**Safari：**
```

→ "添加到主屏幕"
→ 点击"添加"
```

**Chrome：**
```

→ "添加到主屏幕"
→ 点击"添加"
```

现在主屏幕上会出现游戏图标，点击即可游玩！

**增强版特有功能：**
```
1. 打开游戏

4. 允许摄像头权限
```

## 🔧 故障排除

### Q: 网址打不开？

**A**: 检查以下几点：
1. ✅ GitHub Pages是否已启用
2. ✅ 等待30秒-2分钟部署完成
3. ✅ 网址拼写是否正确
4. ✅ 分支是否选对（main或master）

### Q: 显示404错误？

**A**:
1. 确认文件名正确：`zuoyou_hubo_enhanced.html`
2. 检查文件是否在仓库根目录
3. 刷新浏览器缓存（Ctrl+F5）

### Q: iPhone上加载慢？

**A**:
1. 首次加载需要下载（40KB，很快）
2. 之后会缓存，秒开
3. 可以添加到主屏幕，体验更好

### Q: 手势识别不工作？

**A**:
1. 确认使用增强版
2. 点击开启"手势识别模式"
3. 允许摄像头权限
4. 确保光线充足
