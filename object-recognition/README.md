# 物品识别系统 (Object Recognition System)

基于 TensorFlow.js 和 COCO-SSD 模型的智能物品识别系统，支持图片上传和实时摄像头检测。

## 功能特点

- **图片识别**: 支持上传 JPG、PNG、GIF 等格式的图片进行物品识别
- **实时检测**: 通过摄像头进行实时物品检测
- **智能标注**: 自动在图片上绘制边界框和标签
- **中文支持**: 常见物品名称的中文翻译
- **本地处理**: 所有计算在浏览器本地完成，保护隐私
- **80种物品**: 可识别包括人、动物、车辆、家具、食物、电子设备等80种常见物品

## 技术栈

- **TensorFlow.js**: Google 的机器学习框架 JavaScript 版本
- **COCO-SSD**: 预训练的物体检测模型
- **Tailwind CSS**: 现代化的 CSS 框架
- **Canvas API**: 用于绘制检测结果

## 使用方法

### 方式一：直接打开 HTML 文件

1. 直接在浏览器中打开 `index.html` 文件
2. 等待模型加载完成（首次加载可能需要几秒钟）
3. 选择识别方式：
   - **上传图片**: 点击"选择图片"按钮上传本地图片
   - **启动摄像头**: 点击"启动摄像头"按钮进行实时检测

### 方式二：使用本地服务器

推荐使用本地服务器运行，以获得更好的性能：

```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 然后在浏览器访问
# http://localhost:8000
```

## 可识别的物品类别

系统可以识别以下80种物品：

### 人物和动物
人、鸟、猫、狗、马、羊、牛、大象、熊、斑马、长颈鹿

### 交通工具
自行车、汽车、摩托车、飞机、公共汽车、火车、卡车、船

### 日常用品
背包、雨伞、手提包、领带、行李箱、瓶子、酒杯、杯子、叉子、刀、勺子、碗

### 食物
香蕉、苹果、三明治、橙子、西兰花、胡萝卜、热狗、披萨、甜甜圈、蛋糕

### 家具和设备
椅子、沙发、盆栽、床、餐桌、马桶、电视、笔记本电脑、鼠标、遥控器、键盘、手机

### 电器
微波炉、烤箱、烤面包机、水槽、冰箱

### 运动器材
飞盘、滑雪板、运动球、风筝、棒球棒、棒球手套、滑板、冲浪板、网球拍

### 其他
书、时钟、花瓶、剪刀、泰迪熊、吹风机、牙刷、交通信号灯、消防栓、停车标志等

## 性能说明

- **置信度**: 每个检测结果都会显示置信度（0-100%）
  - > 70%: 高置信度（绿色）
  - 50-70%: 中等置信度（黄色）
  - < 50%: 低置信度（橙色）

- **检测速度**:
  - 静态图片: 即时检测
  - 实时摄像头: 根据设备性能，通常可达 10-30 FPS

## 隐私保护

- 所有图片处理和 AI 推理均在浏览器本地完成
- 不会上传任何图片或视频到服务器
- 不收集用户数据
