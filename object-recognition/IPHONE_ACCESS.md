## 🚀 快速开始（3步）

### 步骤 1: 确保设备在同一网络
- iPhone 和运行服务器的电脑必须连接到同一个 WiFi 网络

### 步骤 2: 获取服务器地址
服务器已在端口 8080 运行，您需要知道电脑的IP地址：

**在电脑上查看IP地址：**
- **Windows**:
  ```
  打开命令提示符，输入: ipconfig
  查找 "IPv4 地址"
  ```
- **Mac**:
  ```
  系统偏好设置 → 网络 → 查看 IP 地址
  或终端输入: ifconfig | grep "inet "
  ```
- **Linux**:
  ```
  终端输入: hostname -I
  ```

### 步骤 3: 在 iPhone Safari 中访问
1. 打开 Safari 浏览器
2. 在地址栏输入：`http://你的电脑IP:8080`
   - 例如：`http://192.168.1.100:8080`
   - 或者：`http://21.0.0.50:8080`（如果电脑IP是21.0.0.50）
3. 等待页面加载完成
4. 开始使用物品识别功能！

## 🎯 使用功能

### 1. 图片识别
- 点击"选择图片"按钮
- 从相册选择照片
- 或直接使用相机拍照
- 系统自动识别并标注物品

### 2. 实时摄像头检测
- 点击"启动摄像头"按钮
- **首次使用**: Safari 会请求摄像头权限，点击"允许"
- 后置摄像头自动启动
- 对准物品，系统实时识别
- 点击"停止摄像头"结束检测

## ⚙️ 权限设置

### 允许摄像头访问

**首次访问时：**
1. 点击"启动摄像头"
2. Safari 弹出权限请求
3. 选择"允许"

**如果误点"拒绝"：**
1. 打开 iPhone 设置
2. 向下滚动找到"Safari"
3. 点击"摄像头"
4. 选择"允许"
5. 返回 Safari 刷新页面

## 🔧 故障排除

### 问题：无法访问网站
**解决方案：**
1. 确认 iPhone 和电脑在同一 WiFi
2. 检查 IP 地址是否正确
3. 确认服务器正在运行（端口 8080）
4. 尝试重启服务器
5. 检查防火墙设置，允许端口 8080

### 问题：摄像头无法启动
**解决方案：**
1. 检查 Safari 摄像头权限（见上方"权限设置"）
2. 确保没有其他应用占用摄像头
3. 尝试刷新页面 (下拉页面刷新)
4. 重启 Safari 浏览器
5. 重启 iPhone（极少需要）

### 问题：加载速度慢
**解决方案：**
1. 检查 WiFi 信号强度
2. 首次加载需下载 AI 模型（约 5-10MB）
3. 后续访问会更快（有缓存）
4. 关闭其他占用网络的应用

### 问题：识别不准确
**解决方案：**
1. 确保光线充足
2. 物品在镜头中清晰可见
3. 尝试调整距离和角度
4. 单个物品识别效果最佳
5. 系统可识别 80 种常见物品（见列表）

## 💡 使用技巧

### 获得最佳识别效果
1. **光线**: 确保环境光线充足
2. **距离**: 距离物品 30-100cm 为佳
3. **角度**: 正面拍摄效果最好
4. **背景**: 简洁的背景有助于识别
5. **物品**: 常见物品识别率更高

### 节省电量
1. 图片识别比实时检测更省电
2. 实时检测建议连接充电器
3. 不使用时及时停止摄像头
4. 降低屏幕亮度

### 隐私保护
- ✅ 所有处理在本地完成
- ✅ 不上传任何图片到服务器
- ✅ 不收集用户数据
- ✅ 摄像头完全由您控制
