## 🎉 欢迎使用移动版！

这是专门为iPhone和iPad优化的"左右互搏术"练习程序Web版本！

## 📲 在iPhone上使用的方法

### 方法一：直接在Safari浏览器打开

1. **打开文件**
   - 将 `zuoyou_hubo_mobile.html` 文件发送到你的iPhone
   - 可以通过以下方式：
     - AirDrop发送
     - 上传到云盘（iCloud、百度网盘等）
     - 通过邮件发送给自己
     - 使用文件传输工具

2. **用Safari打开**
   - 在iPhone上找到该文件
   - 点击分享按钮
   - 选择"在Safari中打开"

3. **添加到主屏幕（推荐）**
   - 在Safari中打开文件后
   - 点击底部的"分享"按钮 📤
   - 向下滚动找到"添加到主屏幕"
   - 点击右上角的"添加"
   - 现在你可以像打开App一样使用了！

### 方法二：部署到服务器（推荐）

1. **使用GitHub Pages**
   ```bash
   # 在你的仓库中
   # 1. 将文件推送到main分支
   # 2. 在GitHub仓库设置中启用GitHub Pages
   # 3. 选择main分支作为源
   # 4. 你将获得一个URL，如：https://username.github.io/repo-name/zuoyou_hubo_mobile.html
   ```

2. **在iPhone上访问**
   - 打开Safari浏览器
   - 输入你的GitHub Pages URL
   - 添加到主屏幕以便快速访问

### 方法三：本地服务器（适合开发者）

```bash
# 在项目目录下运行简单的HTTP服务器
python3 -m http.server 8000

# 然后在iPhone的Safari中访问
# http://你的电脑IP地址:8000/zuoyou_hubo_mobile.html
# 注意：iPhone和电脑需要在同一WiFi网络
```

## 💡 使用技巧

### 获得更好体验

1. **添加到主屏幕**
   - 像真正的App一样使用
   - 启动更快，体验更好
   - 全屏显示，沉浸感强

2. **横屏模式**
   - 支持竖屏和横屏
   - 横屏时有更大的显示空间
   - 自动适配屏幕方向

3. **节省电量**
   - 降低屏幕亮度
   - 开启低电量模式
   - 游戏使用纯CSS动画，性能优秀

### 获得更高分数

1. **理解相克系统**
   ```
   拳 → 掌、棍
   掌 → 指、剑
   指 → 拳、内功
   剑 → 拳、棍
   内功 → 掌、剑
   棍 → 指、内功
   ```

2. **选择合适难度**
   - 从初学开始，逐步提升
   - 每个难度都有最佳得分策略

3. **保持平衡**
   - 左右手势均力敌时协调度提升最多
   - 追求平衡而非单方面胜利

## 🔧 故障排除

### 无法打开文件
- ✅ 确保使用Safari浏览器
- ✅ 检查文件是否完整下载
- ✅ 尝试重新下载文件

### 显示不正常
- ✅ 确保iOS系统版本在12.0以上
- ✅ 清除Safari缓存重新打开
- ✅ 尝试横屏/竖屏切换

### 触摸无响应
- ✅ 重新加载页面
- ✅ 检查是否有其他App在后台
- ✅ 重启Safari浏览器

### 动画卡顿
- ✅ 关闭其他后台应用
- ✅ 重启设备
- ✅ 确保不要在低电量模式下使用

## 💬 常见问题

**Q: 可以在安卓手机上使用吗？**
A: 可以！虽然是为iOS优化，但在安卓Chrome浏览器中也能完美运行。

**Q: 需要联网吗？**
A: 不需要！这是纯前端HTML文件，保存到本地后完全离线可用。

**Q: 数据会保存吗？**
A: 当前版本不保存历史记录，每次练习独立计算。未来可能加入本地存储功能。

**Q: 可以自定义招式吗？**
A: 当前版本招式固定为14种经典武功。如需修改，可以编辑HTML文件中的JavaScript代码。

**Q: 为什么有时候感觉不公平？**
A: 游戏加入了随机因素（威力波动±10），模拟真实武功对抗的不确定性。
